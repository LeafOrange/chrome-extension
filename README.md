# chrome-extension
chrome 浏览器效率插件

## 图转文字插件
基于 PaddleOCR PP-OCRv4 的本地离线 OCR 扩展，支持截图识别、文本编辑与复制，全程数据不上传，保护隐私安全。

主要功能：
- 三种截图模式：可视区截图 / 选区截图 / 整页长图截图
- 本地离线 OCR 识别，支持中英混排
- 识别结果可编辑、一键复制
- 本地保存最近 30 条识别历史记录
- 悬浮图标入口，快捷键支持
